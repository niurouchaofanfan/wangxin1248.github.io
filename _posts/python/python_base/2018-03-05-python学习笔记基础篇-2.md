在学习一门语言之前，按照惯例先实现Hello World的输出

在Ubuntu的命令行中输入*python*进入python的交互模式，然后输入如下的代码，回车执行：
``` python
print("Hello World")
```
恭喜你，python入门的第一步就结束了。
