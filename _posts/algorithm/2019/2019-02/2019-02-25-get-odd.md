# 方法一

第一种方法是最常见的方法：模运算

```cpp
if(num%2){
    //奇数
}else{
    //偶数
}
```

# 方法二

第二种方法是：位运算-&

位运算 & 与运算的基本规则：

- 0 & 0 = 0
- 1 & 0 = 0
- 1 & 1 = 1

也就是说只有两个数都为 1 的时候进行与运算的结果才为1。

对于一个十进制的数来说，其转换为二进制的时候是按照**8421**的基本方式来转换的，也就是说对于奇数来说其二进制的最后一位肯定是1，而偶数的二进制的最后一位肯定是0.这样就可以将十进制与1进行与运算来判断这个数是不是偶数。

```cpp
if(num&1){
    //奇数
}else{
    //偶数
}
```
