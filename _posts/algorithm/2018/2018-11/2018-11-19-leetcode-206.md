# 反转链表

## 题目描述

反转一个单链表。

示例:

输入: 1->2->3->4->5->NULL<br/>
输出: 5->4->3->2->1->NULL<br/>
进阶:<br/>
你可以迭代或递归地反转链表。你能否用两种方法解决这道题？

## 解题思路

## 代码-cpp

```cpp
/**
 * Definition for singly-linked list.
 * struct ListNode {
 *     int val;
 *     ListNode *next;
 *     ListNode(int x) : val(x), next(NULL) {}
 * };
 */
class Solution {
public:
    ListNode* reverseList(ListNode* head) {
        ListNode* head1 = NULL;
        while(head != NULL){
            ListNode* a = head->next;
            head->next = head1;
            head1 = head;
            head = a;
        }
        return head1;
    }
};
```
